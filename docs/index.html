EpiCare

A Reinforcement Learning Benchmark for Dynamic Treatment Regimes

About the Project

Welcome to EpiCare, a benchmarking library for reinforcement learning in healthcare applications.

Features

  • A first benchmark for dynamic treatment regimes in healthcare.
  • Reproducibile and configurable: an open source tool conforming to OpenAI Gym standards.
  • Includes realistic costs and patient safety terms to yield a benchmark with realistic difficulty.
